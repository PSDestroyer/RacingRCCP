RCCP AI Assistant offers two methods for accessing the Claude AI: using your own API key for direct access, or using the BoneCracker Games server proxy. This document explains both options in detail to help you choose the best approach for your needs.

Own API Key Mode

What It Is

When you use your own API key, the AI Assistant connects directly to Anthropic's Claude API. All requests go straight from your Unity Editor to Anthropic's servers, with your API key authenticating the connection.

Benefits

Unlimited Requests

There are no request limits when using your own key. You can generate as many configurations as needed without waiting for resets or worrying about quotas.

Direct Connection

Requests go directly to Anthropic without an intermediary, potentially resulting in faster response times.

No Usage Tracking

Your usage is between you and Anthropic. The BoneCracker Games server is not involved in your requests.

Full Model Access

Choose any available Claude model based on your needs and budget.

How to Get an API Key

  1. Visit console.anthropic.com
  2. Create an account or sign in
  3. Navigate to API Keys section
  4. Click "Create Key"
  5. Copy the key (starts with sk-ant-)
  6. Add payment method to your Anthropic account

Important: Keep your API key secure. Never share it or commit it to version control.

Setting Up Your API Key

  1. Open the AI Assistant window
  2. Click the Settings button (gear icon in sidebar)
  3. Find the API Configuration section
  4. Toggle "Use Own API Key" to enabled
  5. Paste your API key in the field
  6. Click "Validate Key" to verify it works

The assistant will test the key by making a minimal API call. If successful, you'll see a green checkmark.

Cost Considerations

Anthropic charges per token for API usage. Typical costs for vehicle configuration:

  • Input tokens: Your prompt + system context (~2,000-4,000 tokens per request)
  • Output tokens: The AI's response (~500-2,000 tokens per request)

Approximate costs per 1,000 tokens (as of 2025):

ModelInputOutput
Haiku 4.5$0.25$1.25
Sonnet 4.5$3$15
Opus 4.5$15$75

A typical vehicle creation request might cost $0.01-0.05 with Haiku, $0.10-0.30 with Sonnet, or $0.50-1.50 with Opus.

Model Selection

When using your own API key, you can choose between three Claude models:

Claude Haiku 4.5 (Fastest, Cheapest)

  • Best for: Quick iterations, simple configurations
  • Response time: 2-5 seconds
  • Cost: ~12x cheaper than Sonnet
  • Quality: Good for straightforward requests

Claude Sonnet 4.5 (Balanced)

  • Best for: Most use cases, balanced quality/cost
  • Response time: 5-15 seconds
  • Cost: Baseline reference
  • Quality: Excellent for complex configurations

Claude Opus 4.5 (Most Capable)

  • Best for: Complex requests, nuanced requirements
  • Response time: 10-30 seconds
  • Cost: ~5x more than Sonnet
  • Quality: Best understanding and output quality

Select your model in Settings > API Configuration > Text Model.

Server Proxy Mode

What It Is

The server proxy routes your AI requests through a BoneCracker Games server. The server handles authentication with Anthropic and manages usage tracking, providing a free tier for evaluation and light usage.

How It Works

  1. Your Unity Editor sends requests to the BoneCracker Games server
  2. The server authenticates your device using a unique token
  3. The server forwards your request to Anthropic's Claude API
  4. The response returns through the server to your Editor
  5. Usage is tracked server-side for rate limiting

Benefits

No API Key Required

Start using the AI assistant immediately without creating an Anthropic account or adding payment information.

Free Tier Included

Every device gets a free allocation of requests:

  • Setup Phase: 400 requests (no daily limit) + 20 per hour burst
  • After Setup: 20 requests per day + 10 per hour burst

Automatic Registration

The first time you make a request, your device is automatically registered. No manual setup required.

Protected API Key

The Anthropic API key is stored securely on the server, not on your machine.

Device Registration

When you first use the server proxy, automatic registration occurs:

  1. The assistant generates a unique device fingerprint
  2. This fingerprint is sent to the server
  3. The server creates a device record and returns a token
  4. The token is stored locally for future requests

The device token identifies your installation for usage tracking. If you reinstall Unity or change machines, you'll get a new device token with fresh usage allowances.

Security Measures

The server proxy implements several security measures:

HMAC Request Signing

Every request is signed with HMAC-SHA256 using your device token. This prevents request forgery and ensures requests come from legitimate installations.

Device Fingerprinting

Your device is identified by a combination of hardware and software characteristics, making it difficult to spoof registrations.

TLS Encryption

All communication uses HTTPS with TLS 1.2, protecting your prompts and responses in transit.

Server-Side Rate Limiting

Usage limits are enforced on the server, not locally. This prevents tampering with usage counters.

Limitations

Rate Limits

Unlike your own API key, the server proxy has request limits (see Usage Limits documentation).

Potential Latency

Requests pass through an additional server, adding some latency compared to direct API access.

Server Availability

If the BoneCracker Games server is down, AI features won't work until it's restored.

When to Use Server Proxy

The server proxy is ideal for:

  • Evaluation: Try the AI assistant before committing to an API key
  • Light Usage: Occasional configuration tasks that fit within free limits
  • Education: Learning how the tool works without cost concerns
  • Testing: Verifying your workflow before scaling up

Switching to Own API Key

If you outgrow the free tier:

  1. Get an Anthropic API key (see above)
  2. Open Settings in the AI Assistant
  3. Toggle "Use Own API Key" to enabled
  4. Enter your API key
  5. Validate the key

Your server proxy registration remains valid if you want to switch back.

Choosing the Right Mode

Choose Own API Key If:

  • You need unlimited requests
  • You're using the tool in production
  • Response time is critical
  • You have an existing Anthropic account
  • You prefer direct API access

Choose Server Proxy If:

  • You're evaluating the tool
  • Your usage is occasional/light
  • You don't want to set up payment
  • You're comfortable with rate limits
  • You want the simplest setup

Advanced Configuration

API Endpoint (Own Key Only)

The default API endpoint is: 


https://api.anthropic.com/v1/messages

This should not be changed unless you're using a custom proxy or testing environment.

Server URL (Server Proxy Only)

The default server URL is configured automatically. Advanced users can override this in settings for testing or custom deployments.

TLS Validation

For development with self-signed certificates, there's a developer option to disable TLS validation. This is dangerous and should only be used for local testing.

Warning: Never disable TLS validation when connecting to production servers.

Troubleshooting

"Invalid API Key"

  • Verify the key starts with sk-ant-
  • Check for leading/trailing whitespace
  • Ensure your Anthropic account has billing set up
  • Regenerate the key if it may have been compromised

"Device Not Registered"

  • Check your internet connection
  • Try clicking "Test Server Connection" in Settings
  • Clear registration and re-register if issues persist

"Rate Limit Exceeded"

  • Wait for the limit to reset (see Usage Limits documentation)
  • Switch to your own API key for unlimited access

Slow Response Times

  • Try a faster model (Haiku instead of Opus)
  • Check your internet connection stability
  • If using server proxy, direct API access may be faster
