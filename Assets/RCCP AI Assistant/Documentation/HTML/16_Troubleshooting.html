This guide covers common issues with RCCP AI Assistant and how to resolve them. Issues are organized by category for easy reference.

Connection Issues

"No response from server"

Symptoms:

  • Request times out
  • No response after extended wait
  • Spinner continues indefinitely

Causes and Solutions:

  1. Internet connectivity
  • Check your internet connection
  • Try loading a website to verify
  • Restart your router if needed
  1. Server downtime
  • The BoneCracker Games server may be temporarily down
  • Wait a few minutes and try again
  • Check for announcements about maintenance
  1. Firewall blocking
  • Unity or the assistant may be blocked by firewall
  • Add Unity to firewall exceptions
  • Check corporate network policies
  1. Timeout too short
  • Increase server timeout in Settings
  • Complex requests need more time
  • Default is 120 seconds

"Device not registered"

Symptoms:

  • Error message about registration
  • Cannot make any requests
  • Settings show no device token

Solutions:

  1. Automatic registration failed
  • Check internet connection
  • Try clicking "Test Server Connection"
  • Wait a moment and try again
  1. Registration stuck
  • Clear registration in Settings
  • Restart Unity
  • Re-attempt request (triggers new registration)

"SSL/TLS Error"

Symptoms:

  • Certificate validation failed
  • HTTPS connection error
  • Security-related error messages

Solutions:

  1. System clock
  • Verify system date and time are correct
  • SSL certificates validate against time
  1. Unity version
  • Ensure Unity 6000.0.49f1 or later
  • Older versions may have SSL issues
  1. Network proxy
  • Corporate proxies can interfere
  • Try from a different network

Rate Limit Issues

"Rate limit exceeded"

Symptoms:

  • Error message about limits
  • Cannot make new requests
  • Usage shows 0 remaining

Solutions:

  1. Daily limit reached
  • Wait until midnight for reset
  • Or switch to your own API key
  1. Hourly burst reached
  • Wait 60 minutes for hourly reset
  • Pace your requests
  1. Setup pool exhausted
  • You've transitioned to free phase
  • Daily limits now apply

Usage not updating

Symptoms:

  • Usage display seems wrong
  • Counter doesn't change after requests

Solutions:

  1. Cached data
  • Refresh usage from Settings
  • Make a request to sync with server
  1. Display lag
  • UI may take a moment to update
  • Wait for next repaint

Vehicle Creation Issues

"Model not eligible"

Symptoms:

  • Eligibility checks fail
  • Cannot proceed to generation
  • Red error icons

Solutions:

  1. Scale issues
  • Check model import settings
  • Scale Factor should produce real-world dimensions
  • A car should be ~4m long, not 4cm or 400m
  1. No separate wheels
  • Wheels must be separate GameObjects
  • Not baked into body mesh
  • Re-export model with separated wheels
  1. Wrong orientation
  • Z should point forward
  • Y should point up
  • Rotate model or use parent to correct

Wheels detected incorrectly

Symptoms:

  • Wrong wheels assigned to positions
  • Wheel detection shows wrong count
  • Assignment preview looks wrong

Solutions:

  1. Naming convention
  • Include "wheel" in wheel GameObject names
  • Helps detection accuracy
  1. Manual assignment
  • Use manual wheel assignment in post-creation
  • Click each position and select correct wheel
  1. Model structure
  • Ensure wheels are direct or near-direct children
  • Deep nesting can confuse detection

Vehicle falls through ground

Symptoms:

  • Vehicle drops through terrain
  • No collision with ground
  • Physics seem broken

Solutions:

  1. Missing colliders
  • Run the Body Colliders wizard
  • Add at least basic colliders
  1. WheelCollider issues
  • Re-run wheel assignment
  • Check WheelCollider radius isn't zero
  1. Ground layer
  • Ensure ground is on a layer that collides with vehicle

API Key Issues

"Invalid API key"

Symptoms:

  • Key validation fails
  • Error message about authentication
  • Requests fail with 401 error

Solutions:

  1. Key format
  • Must start with sk-ant-
  • Check for extra spaces or characters
  • Ensure complete key was copied
  1. Key revoked
  • Key may have been deleted in Anthropic console
  • Generate a new key
  1. Billing issues
  • Anthropic account needs active billing
  • Check console.anthropic.com for account status

Key validation hangs

Symptoms:

  • Validation spinner never completes
  • No success or error message

Solutions:

  1. Connection issue
  • Check internet connectivity
  • API endpoint may be unreachable
  1. Timeout
  • Wait longer (up to 30 seconds)
  • Check console for errors

Window Issues

Window won't open

Symptoms:

  • Menu item does nothing
  • Window doesn't appear
  • Error in console

Solutions:

  1. Compilation errors
  • Check Console for errors
  • Fix any script compilation issues
  • RCCP AI Assistant won't load with errors
  1. Missing dependencies
  • Install Editor Coroutines package
  • Verify RCCP is installed
  1. Corrupted state
  • Delete RCCP_AISettings.asset from Resources
  • Reopen window to regenerate

UI looks wrong

Symptoms:

  • Missing elements
  • Strange layout
  • Garbled text

Solutions:

  1. Style initialization
  • Close and reopen window
  • Styles may not have initialized
  1. Window mode mismatch
  • Try switching window modes in Settings
  • Restart window after changing
  1. Unity version
  • Verify Unity 6000.0.49f1 or later
  • UI may not render correctly in older versions

Animations not working

Symptoms:

  • No smooth transitions
  • Instant changes
  • No visual feedback

Solutions:

  1. Animations disabled
  • Check Settings > Enable Animations
  • Toggle on if off
  1. Force Repaint off
  • Some animations need repaint
  • Try enabling Force Repaint temporarily

Response Issues

AI misunderstands prompt

Symptoms:

  • Response doesn't match request
  • Wrong components modified
  • Unexpected values

Solutions:

  1. Be more specific
  • Add detail to your prompt
  • Mention specific components
  • Use exact terminology
  1. Use appropriate panel
  • Each panel has specialized context
  • Wrong panel may confuse the AI
  1. Simplify request
  • Break complex requests into steps
  • One change at a time

"Unable to parse response"

Symptoms:

  • Error about JSON parsing
  • Response can't be applied
  • Partial or garbled response

Solutions:

  1. Retry request
  • Sometimes AI produces invalid JSON
  • Try again with same prompt
  1. Simplify prompt
  • Complex prompts may produce problematic responses
  • Break into smaller requests
  1. Check raw response
  • Enable Show Raw AI Responses in Settings
  • Look for obvious issues

Response takes too long

Symptoms:

  • Very slow response time
  • Timeout errors
  • Frustrating delays

Solutions:

  1. Choose faster model
  • Use Haiku instead of Opus
  • Significant speed improvement
  1. Simplify prompt
  • Shorter prompts process faster
  • Less context means faster response
  1. Server load
  • Peak times may be slower
  • Try at different times
  1. Use own API key
  • Direct connection may be faster
  • No proxy server hop

Apply Issues

Changes not taking effect

Symptoms:

  • Click Apply but nothing changes
  • Vehicle looks the same
  • No visible difference

Solutions:

  1. Subtle changes
  • Some changes are not visible (internal values)
  • Test by driving, not just looking
  1. Already at target value
  • The vehicle may already have those values
  • Check before/after comparison
  1. Selective apply
  • Check if component groups are toggled on
  • Some groups may be disabled

Undo doesn't work

Symptoms:

  • Can't undo changes
  • Ctrl+Z does nothing
  • Vehicle stuck in new state

Solutions:

  1. Undo stack full
  • Unity has limited undo history
  • May have been pushed out
  1. Scene not saved
  • Save scene before major changes
  • Reload scene to revert
  1. Use Vehicle History
  • Restore from history instead
  • More reliable than Undo for complex changes

Performance Issues

Editor slowdown

Symptoms:

  • Unity becomes sluggish
  • High CPU usage
  • Slow response to inputs

Solutions:

  1. Disable Force Repaint
  • Turn off in Developer Options
  • Reduces unnecessary repaints
  1. Disable animations
  • Turn off in Settings
  • Reduces visual calculations
  1. Close other windows
  • AI Assistant can be intensive
  • Reduce editor overhead

Large response lag

Symptoms:

  • Delay when receiving large responses
  • UI freezes briefly
  • Slow to render response

Solutions:

  1. Normal behavior
  • Large JSON takes time to parse and render
  • Be patient
  1. Collapse sections
  • Collapsed JSON renders faster
  • Expand only what you need

Getting Help

If you can't resolve an issue:

  1. Check Console
  • Look for error messages
  • Note exact error text
  1. Enable Verbose Logging
  • Get detailed debug information
  • Helps identify the problem
  1. Document the Issue
  • What you were doing
  • What you expected
  • What happened instead
  • Any error messages
  1. Contact Support
  • Visit bonecrackergames.com
  • Provide documentation
  • Include Unity version and RCCP version
