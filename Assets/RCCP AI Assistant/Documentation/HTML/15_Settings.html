The Settings panel provides access to all configuration options for RCCP AI Assistant. This document covers every setting, what it does, and when to adjust it.

Accessing Settings

Open Settings by:

  • Clicking the gear icon in the sidebar
  • Pressing the Settings button at the bottom of the sidebar
  • Using keyboard shortcut (if configured)

API Configuration

Use Own API Key

Toggle between API modes:

Enabled: Use your personal Anthropic API key

  • Direct connection to Claude API
  • Unlimited requests (pay per use)
  • Requires valid API key entered below

Disabled: Use BoneCracker Games server proxy

  • No API key needed
  • Free tier with usage limits
  • Automatic device registration

API Key Field

Enter your Anthropic API key:

  • Starts with sk-ant-
  • Stored securely in EditorPrefs
  • Not visible by default (toggle eye icon to show)

Validate Key Button

Test your API key:

  • Makes a minimal API call
  • Confirms key is valid
  • Shows success or error message

Server Connection Test

Test server proxy connection:

  • Verifies server is reachable
  • Checks device registration
  • Confirms authentication works

API Endpoint (Advanced)

The API endpoint URL:

  • Default: https://api.anthropic.com/v1/messages
  • Only change for custom setups
  • Leave default for normal use

Model Selection

Text Model

Choose the Claude model for text requests:

Haiku 4.5 (Fastest, Cheapest)

  • Best for: Quick iterations, simple requests
  • Response time: 2-5 seconds
  • Cost: ~12x cheaper than Sonnet

Sonnet 4.5 (Balanced)

  • Best for: Most use cases
  • Response time: 5-15 seconds
  • Cost: Baseline reference

Opus 4.5 (Most Capable)

  • Best for: Complex requests
  • Response time: 10-30 seconds
  • Cost: ~5x more expensive than Sonnet

Vision Model

Choose the model for vision-based light detection:

  • Same options as text model
  • Vision requests are more expensive
  • Sonnet recommended for vision tasks

Cost Indicator

Shows relative cost multiplier:

  • Haiku: 0.08x
  • Sonnet: 1.0x
  • Opus: 5.0x

UI Settings

Window Mode

Choose how the window behaves:

Utility Window

  • Fixed size (800x850 pixels)
  • Floats above other windows
  • Cannot be docked

Dockable Window

  • Resizable
  • Can dock to editor layout
  • Integrates with your workspace

Note: Changing this requires restarting the window.

Show Welcome on Startup

When enabled:

  • Welcome panel appears on first open
  • Provides quick-start guidance
  • Disable after you're familiar with the tool

Quick Prompt Count

Number of quick prompt buttons to show (0-25):

  • 0: Hide quick prompts entirely
  • 5-10: Moderate selection
  • 15-25: Full variety

Quick prompts shuffle randomly from the available pool.

Animation Settings

Enable Animations

Toggle UI animations:

Enabled:

  • Smooth panel transitions
  • Response fade-in effects
  • Status message fading
  • Processing pulse indicator

Disabled:

  • Instant transitions
  • No visual effects
  • Slightly better performance

Animation Speed

Adjust animation speed (0.5x - 2.0x):

  • 0.5x: Slow, deliberate animations
  • 1.0x: Normal speed
  • 2.0x: Fast, snappy animations

Prompt Settings

Global Prefix

Text added before every prompt:

  • Applied to all panels
  • Useful for consistent instructions
  • Default: "RESPOND WITH ONLY JSON..."

Global Suffix

Text added after every prompt:

  • Applied to all panels
  • Useful for closing instructions

Max Prompt Length

Maximum characters allowed in prompt input:

  • Default: 1000 characters
  • Increase for longer descriptions
  • Helps prevent overly verbose prompts

Send on Enter

Toggle Enter key behavior:

Enabled:

  • Enter sends the prompt
  • Shift+Enter inserts new line

Disabled:

  • Enter inserts new line
  • Ctrl+Enter sends the prompt

Auto-Apply (Vehicle Creation)

Toggle automatic application of generated configurations:

Enabled:

  • After AI generates a configuration, it's applied immediately
  • Skips the review step for faster workflow
  • Text area shows orange border as visual indicator
  • Best for: Rapid prototyping, experienced users

Disabled (Default):

  • Generated configurations go to the Review Panel
  • You review before/after values before applying
  • Best for: Careful review, learning, production work

Note: Auto-Apply only affects the Vehicle Creation panel. Other panels always show the review step.

Keyboard Shortcuts

Enable Shortcuts

Toggle keyboard shortcuts:

  • When enabled, shortcuts are active
  • When disabled, only standard keys work

Shortcut Reference

ShortcutAction
Enter/Ctrl+EnterSend prompt (depends on setting)
Shift+EnterNew line
EscapeClose panel/overlay

Developer Options

Note: These options are for debugging and development. Enable Developer Mode to access them.

Developer Mode

Enables advanced options:

  • Shows additional controls
  • Exposes debugging features
  • Not needed for normal use

Verbose Logging

When enabled:

  • Logs detailed information to Console
  • Shows API requests and responses
  • Useful for troubleshooting

Force Repaint

When enabled:

  • Window repaints every frame
  • Higher CPU usage
  • Useful for animation debugging

Save Vision Debug Screenshots

When enabled:

  • Saves screenshots used for vision detection
  • Stored in project folder
  • Useful for debugging detection issues

Vision Detection Settings

Configure how vision-based light detection works:

Orthographic Zoom Multiplier

Adjusts the camera zoom for vehicle screenshots:

  • Higher values = closer zoom (more detail)
  • Lower values = wider view (more context)
  • Default: 1.0
  • Adjust if light detection seems inaccurate

Screenshot Resolution

The resolution of captured screenshots:

  • Higher = more accurate detection, larger file
  • Default: 512 pixels
  • Increase for complex vehicles with many lights

Show Raw AI Responses

When enabled:

  • Shows unprocessed JSON responses
  • Bypasses structured view
  • Useful for debugging parsing

Show Request/Response Times

When enabled:

  • Displays timing information
  • Shows how long requests take
  • Useful for performance analysis

Disable TLS Validation

DANGEROUS - Testing Only

When enabled:

  • Disables SSL certificate validation
  • ONLY for local development with self-signed certs
  • Never use with production servers
  • Security risk

Usage Information

This section displays your current usage (when using server proxy):

Current Tier

Shows your usage tier:

  • Setup Phase: Using initial pool
  • Free Phase: Using daily allowance
  • Own API Key: Unlimited

Usage Statistics

Depending on tier:

  • Setup pool remaining/total
  • Daily usage/limit
  • Hourly burst usage/limit
  • Time until reset

Total Requests

Lifetime request count across all sessions.

Last Sync

When usage data was last updated from server.

Prompt Assets

Advanced section for managing prompt assets:

View Prompt Assets

Shows all configured prompt assets:

  • One per panel type
  • Click to open in Inspector

Refresh Prompt Assets

Reloads prompt assets from disk:

  • Use if assets were modified externally
  • Forces re-read of all prompts

Edit Prompts (Advanced)

Direct editing of prompt assets:

  • System prompts
  • Example prompts
  • Panel configuration

Note: Modifying prompts requires understanding of the AI system. Incorrect prompts may produce poor results.

Resetting Settings

Reset to Defaults

Available reset options:

Reset UI Preferences

  • Window mode
  • Animation settings
  • Quick prompt count
  • Keyboard settings

Reset Developer Options

  • Verbose logging
  • Debug options
  • All dev-only settings

Reset All

  • Everything including API key
  • Fresh start
  • Requires re-configuration

Note: Reset operations cannot be undone.

Sharing Settings

To share settings between team members:

  • RCCP_AISettings asset can be version controlled
  • API keys should NOT be shared
  • EditorPrefs are per-user

Backup Considerations

Important settings to preserve:

  • API key (store separately, securely)
  • Custom prompt modifications
  • RCCP_AISettings asset customizations

Common Configurations

For Evaluation

  • Use server proxy (no API key)
  • Sonnet model for quality
  • Animations enabled
  • Default prompt settings

For Production

  • Use own API key
  • Haiku for speed, Sonnet for quality
  • Consider disabling animations for performance
  • Document any prompt customizations

For Debugging

  • Enable developer mode
  • Enable verbose logging
  • Show raw AI responses
  • Save vision debug screenshots
