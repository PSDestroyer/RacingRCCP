Ask Mode allows you to ask questions and get explanations from the AI without making any changes to your vehicle. This is separate from Configure mode, which generates settings you can apply.

Understanding the Two Modes

RCCP AI Assistant has two distinct interaction modes for every panel:

Configure Mode (Default)

Configure mode is the standard workflow:

  • You describe what you want to change
  • AI returns JSON configuration
  • You review the proposed changes
  • You apply changes to your vehicle

The button appears orange when active, and the Generate button says "Generate".

Ask Mode

Ask mode is for questions and learning:

  • You ask a question or request an explanation
  • AI returns plain text response
  • No configuration changes are proposed
  • Nothing is applied to your vehicle

The button appears blue when active, and the Generate button says "Ask".

Switching Between Modes

At the top of the prompt input area, you'll see two buttons: 


[ Configure ]  [ Ask ]

Click the button for the mode you want to use:

  • Configure - For making changes (returns settings)
  • Ask - For getting answers (returns advice only)

A hint below the buttons confirms your current mode:

  • "Returns settings you can review and apply"
  • "Returns advice only (no changes)"

Your mode preference is remembered between sessions.

When to Use Ask Mode

Learning About Vehicle Physics

Ask questions to understand how things work: 


What does the differential torque split percentage mean?



How do ABS, ESP, and TCS work together?



Explain how steering curves affect high-speed handling.

Getting Recommendations

Ask for advice before making changes: 


What power level is appropriate for a family sedan?



Should I use FWD, RWD, or AWD for a rally car?



What suspension stiffness works best for drifting?

Understanding Current Configuration

Ask about your vehicle's current state: 


Why might my vehicle understeer in corners?



What could cause my car to flip over easily?



Is 500hp too much for a lightweight sports car?

Comparing Options

Get explanations of trade-offs: 


What are the pros and cons of automatic vs manual transmission?



How does lowering center of mass affect handling?



What's the difference between soft and stiff suspension?

Ask Mode Response Format

When you use Ask Mode, the response appears differently:

No Review Panel

There's no structured review panel with before/after values. Instead, you see a conversational response.

No Apply Buttons

There are no Apply All, Apply Selected, or Discard buttons because there's nothing to apply.

Text-Based Response

The AI provides a readable explanation rather than technical JSON: 


The differential torque split determines how engine power

is distributed between the front and rear axles in an AWD

vehicle. A 50/50 split sends equal power to both axles,

while a 30/70 split sends more power to the rear...

Clear Button

A "Clear" button dismisses the response when you're done reading.

Example Conversations

Understanding a Component

Prompt (Ask Mode): 


What does the extremumValue in tire friction curves mean?

Response: 


The extremumValue in tire friction curves represents the peak

grip force available before the tire starts to slide. It's the

maximum coefficient of friction at the optimal slip ratio.



A higher extremumValue means more grip at the peak. Typical

values range from 1.0 (standard tires) to 1.5+ (racing slicks).



The extremumSlip value paired with it determines how much wheel

slip occurs at this peak grip point...

Getting Advice

Prompt (Ask Mode): 


My drift car keeps spinning out instead of sliding. What should I adjust?

Response: 


If your drift car spins out instead of maintaining a controlled

slide, consider these adjustments:



1. Increase front grip relative to rear - this helps maintain

   steering control while the rear slides



2. Enable counter-steer assistance - this helps catch slides

   before they become spins



3. Reduce rear tire grip gradually - too little rear grip

   causes snap oversteer



4. Check your ESP settings - even light ESP can fight your

   drift attempts



5. Adjust weight distribution - more rear weight can help

   but too much causes instability...

Clarifying Before Configuring

Prompt (Ask Mode): 


What's a realistic power-to-weight ratio for a sports car?

Response: 


Realistic sports car power-to-weight ratios vary by class:



Entry sports cars: 100-150 hp per ton

  Examples: Mazda MX-5, Toyota 86



Performance sports cars: 150-250 hp per ton

  Examples: Porsche Cayman, BMW M2



Supercars: 300-400 hp per ton

  Examples: Ferrari 488, Lamborghini Huracan



Hypercars: 500+ hp per ton

  Examples: McLaren P1, Bugatti Chiron



For a balanced, fun-to-drive experience, aim for 150-200 hp

per ton. Higher ratios require more skill to control...

Best Practices

Use Ask Mode First

When you're unsure about a configuration:

  1. Switch to Ask Mode
  2. Ask for advice or explanation
  3. Switch back to Configure Mode
  4. Make informed changes

Ask Specific Questions

Vague questions get vague answers. Compare:

Less effective: 


How do I make my car better?

More effective: 


How can I reduce body roll in corners without making

the ride too harsh?

Reference Your Vehicle

Mention relevant details: 


My 400hp RWD sports car has trouble putting power down

from a standstill. What settings should I look at?

Ask Follow-Up Questions

If the first answer isn't enough: 


You mentioned adjusting the torque curve. Can you explain

what shape works best for linear acceleration feel?

Limitations of Ask Mode

No Configuration Changes

Ask Mode cannot modify your vehicle. Even if you phrase a request as a question: 


Can you increase my engine power to 300hp?

The AI will explain what that change would involve rather than making the change. Switch to Configure Mode to actually apply changes.

No Before/After Comparison

Since no changes are proposed, there's no comparison view. If you want to see what values would change, use Configure Mode.

Counts Against Request Quota

Ask Mode requests count toward your usage limits just like Configure Mode requests. Each question uses one request from your daily allowance.

Panel Context Still Applies

The AI's knowledge is still focused on the current panel's domain. Asking about lights while in the Behaviors panel may get a less detailed answer than asking in the Lights panel.

Integration with Other Features

Prompt History

Ask Mode prompts are saved to Prompt History with an "Informational" flag. You can:

  • Search past questions
  • Review previous answers
  • Identify which prompts were questions vs configurations

Vehicle History

Ask Mode responses are NOT saved to Vehicle History because no changes are made to the vehicle.

Quick Prompts

Quick prompt buttons are designed for Configure Mode. In Ask Mode, write your own questions rather than using quick prompts.

Tips for Effective Questions

Be Specific About Context

Include relevant details:

  • Vehicle type you're working with
  • Current problem you're experiencing
  • What you've already tried
  • Your intended use case

Ask One Thing at a Time

Instead of: 


How do ABS, ESP, and TCS work and what settings should I use?

Ask separately: 


How does ABS work in RCCP?

Then: 


What ABS strength is good for a track car?

Use Technical Terms When You Know Them

If you know the component name: 


What does the RCCP_Differential's viscousLocking parameter do?

Ask for Recommendations

The AI can suggest starting points: 


What are good default values for a pickup truck's suspension?
