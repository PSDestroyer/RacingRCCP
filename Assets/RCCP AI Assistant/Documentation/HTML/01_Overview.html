What is RCCP AI Assistant?

RCCP AI Assistant is an AI-powered editor tool that integrates with Realistic Car Controller Pro (RCCP) to streamline the process of creating, configuring, and tuning vehicles in Unity. Instead of manually adjusting dozens of component properties across multiple inspectors, you describe what you want in natural language and the AI generates the appropriate configuration.

The assistant transforms the traditionally technical and time-consuming process of vehicle setup into a conversational workflow. Whether you want to create a high-performance sports car, a stable family sedan, or a drift-ready race car, you simply describe your vision and the AI handles the technical details.

Key Features and Capabilities

AI-Powered Vehicle Creation

Transform raw 3D models into fully functional RCCP vehicles with a single natural language description. The AI analyzes your model's mesh structure, identifies wheels, and generates appropriate physics configurations based on your description.

Intelligent Vehicle Customization

Modify existing RCCP vehicles by describing what you want to change. The AI understands context and can adjust multiple related properties to achieve your desired behavior.

Specialized Configuration Panels

Eight purpose-built panels for different aspects of vehicle configuration:

  • Vehicle Creation - Convert 3D models to RCCP vehicles
  • Vehicle Customization - General modifications to existing vehicles
  • Behaviors - Driving styles, stability systems, steering response
  • Wheels - Tire friction, suspension, alignment settings
  • Audio - Engine sound configuration
  • Lights - Headlights, taillights, indicators with vision-based detection
  • Damage - Deformation, detachable parts, collision response
  • Diagnostics - Local validation checks (no AI required)

Before/After Change Preview

Review proposed changes before applying them. See exactly what values will change and selectively apply modifications.

Batch Operations

Create or customize multiple vehicles at once using a single description, perfect for fleet setups or consistent configurations across vehicle variants.

Prompt History

Full history of all prompts and responses, searchable and filterable, with the ability to re-apply configurations to new vehicles.

How It Integrates with RCCP

RCCP AI Assistant works alongside RCCP (Realistic Car Controller Pro), the comprehensive vehicle physics system by BoneCracker Games. The AI assistant doesn't replace RCCP - it provides an intelligent interface for configuring RCCP components.

When you create or customize a vehicle through the AI assistant, it modifies the same components you would configure manually:

  • RCCP_CarController - Main vehicle controller
  • RCCP_Engine - Engine power and RPM curves
  • RCCP_Gearbox - Transmission settings
  • RCCP_Differential - Power distribution
  • RCCPAxles / RCCPAxle - Axle configurations
  • RCCP_Stability - ABS, ESP, TCS systems
  • RCCP_Lights - Vehicle lighting
  • RCCP_Audio - Sound configuration
  • And more...

The AI understands RCCP's component architecture and generates configurations that work within its physics model.

Who Is It For?

Game Developers

Rapidly prototype vehicle configurations without deep knowledge of vehicle physics. Get playable vehicles quickly and iterate on handling feel using natural language.

Technical Artists and Designers

Focus on the creative aspects of vehicle design while the AI handles the technical configuration. Describe the driving experience you envision and let the AI translate that into component settings.

Indie Developers and Hobbyists

Get professional-quality vehicle setups without extensive experience in vehicle physics. The AI provides sensible defaults and expert-level configurations based on your descriptions.

Studios with Large Vehicle Rosters

Configure multiple vehicles consistently and efficiently. Batch operations ensure uniformity across vehicle variants while prompt history enables easy reproduction of configurations.

What You Can Do With It

Create New Vehicles

  • Start with any 3D car model (properly scaled, with separate wheel meshes)
  • Describe the vehicle type and behavior you want
  • The AI generates complete RCCP configuration
  • Post-creation wizards help assign wheels and add colliders

Customize Existing Vehicles

  • Select any vehicle with an RCCP_CarController
  • Describe the modifications you want
  • Preview changes before applying
  • Apply selectively or all at once

Tune Vehicle Behavior

  • Adjust driving dynamics through natural language
  • Configure stability systems (ABS, ESP, TCS)
  • Set up steering response curves
  • Define speed limiters and behavior presets

Configure Vehicle Components

  • Set up tire friction for different conditions
  • Configure suspension for handling characteristics
  • Adjust audio for engine sound profiles
  • Set up lights with automatic position detection
  • Configure damage systems and detachable parts

The AI-Powered Workflow

The RCCP AI Assistant follows a simple three-step workflow:

Step 1: Select Target

Choose what you want to work with - either a raw 3D model for vehicle creation or an existing RCCP vehicle for customization. The assistant validates your selection and shows relevant information about the model or vehicle.

Step 2: Describe What You Want

Write a natural language description of what you want to achieve. Be specific about performance characteristics, handling feel, or behavior patterns. The AI interprets your description and generates appropriate configurations.

Step 3: Review and Apply

Before any changes are made, review the proposed configuration. See before/after comparisons of values that will change. Apply all changes at once or selectively toggle individual component groups.

This workflow keeps you in control while leveraging AI to handle the technical complexity of vehicle configuration.

Getting Started

To begin using RCCP AI Assistant:

  1. Ensure you have RCCP (Realistic Car Controller Pro) installed
  2. Install the Editor Coroutines package (required dependency)
  3. Open the assistant: Tools > BoneCracker Games > RCCP AI Assistant > Open Assistant
  4. Configure your API access (own key or server proxy)
  5. Start creating or customizing vehicles

For detailed setup instructions, see the Installation documentation.
