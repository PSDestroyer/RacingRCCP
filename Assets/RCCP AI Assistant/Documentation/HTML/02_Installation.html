System Requirements

Unity Version

RCCP AI Assistant requires Unity 6000.0.49f1 or later (Unity 6). Earlier versions of Unity are not supported due to API dependencies and editor features introduced in Unity 6.

Required Packages

Editor Coroutines (Required)

The assistant uses editor coroutines for asynchronous API communication. Install version 1.0.1 or later from the Package Manager:

  1. Open Window > Package Manager
  2. Click the + button and select "Add package by name"
  3. Enter: com.unity.editorcoroutines
  4. Click Add

Alternatively, add this line to your Packages/manifest.json: 


"com.unity.editorcoroutines": "1.0.1"

RCCP (Realistic Car Controller Pro) (Required)

RCCP AI Assistant is designed to work with RCCP. Ensure you have RCCP installed and configured in your project before using the AI assistant.

Internet Connection

An active internet connection is required for AI functionality. The assistant communicates with either the Anthropic Claude API directly (if using your own key) or through the BoneCracker Games server proxy.

Installation Steps

From Unity Asset Store

  1. Purchase and Download
  • Purchase RCCP AI Assistant from the Unity Asset Store
  • Open the Package Manager (Window > Package Manager)
  • Select "My Assets" from the dropdown
  • Find "RCCP AI Assistant" and click "Download"
  • Click "Import" to add it to your project
  1. Import Settings
  • Import all files when prompted
  • The package will be installed to Assets/RCCP AI Assistant/
  1. Verify Installation
  • Check that the menu item appears: Tools > BoneCracker Games > RCCP AI Assistant
  • You should see both "Open Assistant" and other submenu items

Manual Installation

If you have the package files directly:

  1. Copy the RCCP AI Assistant folder to your project's Assets directory
  2. Wait for Unity to compile scripts
  3. Verify the menu items appear under Tools > BoneCracker Games

First-Time Setup

Opening the Window

Open the AI Assistant window:

  • Tools > BoneCracker Games > RCCP AI Assistant > Open Assistant

Automatic Asset Creation

On first launch, the assistant automatically creates required assets if they don't exist:

  • RCCP_AISettings.asset - Main settings file (created in Assets/RCCP AI Assistant/Resources/)
  • Prompt Assets - Panel-specific AI prompts (created in Assets/RCCP AI Assistant/Resources/Prompts/)
  • Component Defaults - RCCP default values reference (created in Assets/RCCP AI Assistant/Resources/)

These assets are ScriptableObjects that store configuration. You don't need to modify them directly - the Settings panel in the assistant window provides access to all configurable options.

Welcome Panel

The first time you open the assistant, a Welcome panel appears with:

  • Brief overview of features
  • API configuration options
  • Quick start guidance

You can dismiss this panel and access it later from the Settings if needed.

Folder Structure

After installation, your project will have the following structure: 


Assets/

  RCCP AI Assistant/

    Documentation/         <- Documentation files (this content)

    Editor/               <- Editor scripts (core functionality)

    Resources/

      RCCP_AISettings.asset    <- Main settings

      RCCP_AIComponentDefaults.asset <- RCCP defaults reference

      Prompts/            <- AI prompt assets for each panel

    Scenes/               <- Demo scene (optional)

Key Files

FilePurpose
RCCP_AISettings.assetStores API endpoint, model selection, max tokens, global prompts
RCCP_AIComponentDefaults.assetStores extracted default values from RCCP components
Prompts/*.assetIndividual prompt configurations for each panel

Verification Steps

After installation, verify everything is working:

Check Menu Items

  1. Open Tools > BoneCracker Games
  2. You should see:
  • RCCP AI Assistant submenu with "Open Assistant"
  • Documentation Viewer (for viewing this documentation)

Check Console

Open the Console window (Window > General > Console) and look for:

  • No error messages related to RCCP AI Assistant
  • If you see errors about missing assemblies or types, ensure Editor Coroutines is installed

Check Settings Asset

  1. Navigate to Assets/RCCP AI Assistant/Resources/ in the Project window
  2. Select RCCP_AISettings
  3. In the Inspector, verify:
  • Prompts array is populated with panel assets
  • API endpoint is set (default: https://api.anthropic.com/v1/messages)

Test Window Opening

  1. Open Tools > BoneCracker Games > RCCP AI Assistant > Open Assistant
  2. The window should open without errors
  3. You should see either:
  • Welcome panel (first launch)
  • Main interface with sidebar and panels

Troubleshooting Installation Issues

"The type or namespace 'EditorCoroutines' could not be found"

The Editor Coroutines package is not installed. Install it from Package Manager as described above.

Menu items don't appear

  1. Check Console for compilation errors
  2. Try Assets > Reimport All
  3. Restart Unity

Settings asset is null or missing prompts

  1. Delete Assets/RCCP AI Assistant/Resources/RCCP_AISettings.asset
  2. Reopen the AI Assistant window - it will recreate missing assets

Window opens but shows warnings

If you see "Settings not found" or similar warnings:

  1. Close the AI Assistant window
  2. Run Assets > Reimport All
  3. Reopen the window

RCCP components not recognized

Ensure RCCP (Realistic Car Controller Pro) is properly installed and has no compilation errors before using the AI Assistant.

Updating RCCP

When you update RCCP to a new version, component defaults may change. To sync the AI assistant with the new defaults:

  1. Open Tools > BoneCracker Games > RCCP AI Assistant > Extract Component Defaults
  2. This updates RCCP_AIComponentDefaults.asset with current RCCP values
  3. The AI will now use the correct default values when generating configurations

Running the defaults extractor ensures the AI has accurate information about RCCP's current state.
