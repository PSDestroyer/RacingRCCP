QUICK START
-----------

1. The Setup Wizard should open automatically after importing this package.

   If it doesn't, open it via:
   Tools > BoneCracker Games > RCCP AI Assistant > Setup Wizard

2. Follow the wizard steps:
   - Welcome: Overview of the installation
   - Dependencies: Verify RCCP is installed, install Editor Coroutines
   - API Info: Learn how to get a Claude API key
   - Import: Install the main RCCP AI Assistant package
   - Complete: Start using the AI Assistant!


REQUIREMENTS
------------

- Unity 6 (6000.0+)
- Realistic Car Controller Pro (must be installed first)
- Editor Coroutines package (auto-installed by wizard)


WHAT'S INCLUDED
---------------

This installer package contains:
- Setup wizard for guided installation
- Embedded main package (RCCP_AI_Complete.unitypackage)
- Dependency checker and auto-installer

After installation, you'll have:
- AI-powered vehicle creation wizard
- Natural language vehicle customization
- Behavior preset configuration
- Wheel, audio, lights, and damage panels
- Vision-based light detection
- Complete documentation


CLAUDE API KEY
--------------

The RCCP AI Assistant uses Claude AI for natural language processing.

Free Tier (No API key needed):
- 400 setup requests to get started
- Then 25 requests per day
- Uses our proxy server, no account needed

For unlimited requests, you can use your own API key from Anthropic.

To get an API key:
1. Visit https://console.anthropic.com/
2. Create an account or sign in
3. Navigate to API Keys
4. Create a new API key

You can configure your API key in the Settings panel after installation.
